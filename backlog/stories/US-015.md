# US-015: Filtrar historico por intervalo de datas

**Feature:** FEAT-005
**Status:** Backlog
**Estimativa:** P
**Criado em:** 2026-06-06

---

## User Story
**Como** medico logado
**Quero** filtrar meu historico por periodo de tempo
**Para** encontrar registros de uma semana ou mes especifico

## Criterios de Aceitacao

```gherkin
Cenario 1: Filtro por data de inicio e fim
  DADO que estou no historico unificado
  QUANDO seleciono uma data de inicio e uma data de fim
  ENTAO apenas registros dentro desse intervalo sao exibidos

Cenario 2: Data de inicio posterior a data de fim
  DADO que seleciono uma data de inicio posterior a data de fim
  ENTAO vejo uma mensagem de validacao: "A data de inicio deve ser anterior a data de fim"

Cenario 3: Filtro por atalho (esta semana, este mes)
  DADO que estou nos filtros do historico
  QUANDO clico em "Esta semana"
  ENTAO o intervalo de datas e preenchido automaticamente com os limites da semana atual
```

## Notas Tecnicas

- Pode ser implementado como filtro client-side para simplicidade inicial
- Atalhos recomendados: "Hoje", "Esta semana", "Este mes", "Ultimos 3 meses"

## Dependencias

- US-012: timeline unificada precisa existir

## Criterios de Rejeicao

- Aceitar data de inicio posterior a data de fim sem validacao
- Nao resetar os filtros ao clicar em "Limpar filtros"
