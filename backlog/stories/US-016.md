# US-016: Ver grafico de evolucao de peso ao longo do tempo no perfil do paciente

**Feature:** FEAT-006
**Status:** Backlog
**Estimativa:** M
**Criado em:** 2026-06-06

---

## User Story
**Como** medico logado
**Quero** ver um grafico de linha com o peso do paciente ao longo do tempo
**Para** identificar tendencias de ganho ou perda de peso de forma visual

## Criterios de Aceitacao

```gherkin
Cenario 1: Grafico exibido com 3 ou mais medicoes
  DADO que um paciente tem 3 ou mais registros de peso em datas diferentes
  QUANDO abro o perfil desse paciente
  ENTAO vejo um grafico de linha com peso (kg) no eixo Y e data no eixo X

Cenario 2: Lista textual com menos de 3 medicoes
  DADO que um paciente tem menos de 3 registros de peso
  QUANDO abro o perfil desse paciente
  ENTAO vejo as medicoes como lista de texto (sem grafico)

Cenario 3: Ponto no grafico com tooltip
  DADO que estou vendo o grafico de peso
  QUANDO passo o mouse sobre um ponto do grafico
  ENTAO vejo um tooltip com data e valor exato do peso
```

## Notas Tecnicas

- Dados disponiveis via `GET /api/patients/{id}` no campo `metrics`
- Filtrar apenas entradas com `weight_kg != null` para o grafico de peso
- Ordenar por `date` antes de renderizar
- Biblioteca sugerida: Recharts (leve, compativel com React)

## Dependencias

- Nenhuma dependencia de outras user stories.

## Criterios de Rejeicao

- Renderizar grafico com apenas 1 ponto (nao faz sentido visual)
- Nao ter tooltip nos pontos
- Crashar se `metrics` for array vazio
