# US-001: Ver lista de casos clinicos no historico

**Feature:** FEAT-001
**Status:** Backlog
**Estimativa:** M
**Criado em:** 2026-06-06

---

## User Story
**Como** medico logado
**Quero** ver meus casos clinicos na HistoryPanel
**Para** acessar laudos de casos completos que analisei anteriormente

## Criterios de Aceitacao

```gherkin
Cenario 1: Historico com casos salvos
  DADO que estou logado e tenho casos clinicos salvos
  QUANDO abro a HistoryPanel
  ENTAO vejo os casos clinicos listados com titulo, data e especialidade

Cenario 2: Historico sem casos
  DADO que estou logado e nao tenho casos clinicos salvos
  QUANDO abro a HistoryPanel
  ENTAO vejo uma mensagem "Nenhum caso clinico encontrado"

Cenario 3: Erro de rede ao carregar casos
  DADO que estou logado
  QUANDO a requisicao GET /api/cases falha por erro de rede
  ENTAO vejo uma mensagem de erro com opcao de tentar novamente
```

## Notas Tecnicas

- Consumir `GET /api/cases` (endpoint ja existe no backend)
- Response schema: `{ id, title, chief_complaint, created_at, analyzed_at, patient_id, exams_summary }`
- Normalizar para o mesmo formato que ExamRecord antes de renderizar
- Diferenciar visualmente (badge "Caso Clinico" vs "Quick Analysis")

## Dependencias

- Nenhuma dependencia de outras user stories.

## Criterios de Rejeicao

- Mostrar casos de outros usuarios
- Travar a UI durante o carregamento sem spinner
- Misturar casos e quick analyses sem diferenciacao visual
